# COL215 Assignments Repository

This repository contains the software and hardware assignment submissions for **COL215: Digital Logic and System Design** at IIT Delhi, Semester I 2024-25.

The repository has been documented and organized around assignment-level README files. Each assignment README explains the problem being solved, the implementation approach, key design decisions, tradeoffs, verification flow, and relevant performance or hardware metrics.

## Repository Index

| Section | Description |
| --- | --- |
| [Software Assignments](./SW%20Assignments/README.md) | Algorithmic placement and timing-analysis assignments implemented in Python. |
| [Hardware Assignments](./HW%20Assignments/README.md) | VHDL/FPGA assignments targeting the Basys 3 board. |
| [Final Submissions](./2023CS50334_2023CS10106) | Submitted archives, reports, source files, and supporting artifacts. |
| [VHDL Resources](./VHDL_Resources) | Shared/reference VHDL material used during the hardware assignments. |

## Assignment READMEs

### Software

| Assignment | Focus |
| --- | --- |
| [SW Assignment 1](./SW%20Assignments/SW_Assignment_1/README.md) | Rectangular gate packing and bounding-box optimization. |

| [SW Assignment 3](./SW%20Assignments/SW_Assignment_3/README.md) | Timing-aware gate placement with critical-path analysis and randomized placement search. |

### Hardware

| Assignment | Focus |
| --- | --- |
